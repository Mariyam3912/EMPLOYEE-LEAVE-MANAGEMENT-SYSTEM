Employee Leave Management System
Introduction:
The aim of the project is to develop an online centralized application to automate the workflow of leave
applications and their approvals.
Programming language: Python
Framework: Django
Database: MySql


**Instructions**
(INSIDE src)
- Install the Requirements: pip install -r requirements.txt
(if some modules fails to get installed,
try installing that particular module without version)
- Then, make database migrations: python manage.py makemigrations
- python manage.py migrate
- And finally, run the application: python manage.py runserver

For Admin Account, please create one with superuser!
